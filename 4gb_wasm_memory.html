To detect if a Wasm user agent supports 4GB WebAssembly.Memory sizes (instead of the max 2GB that many user agents were limited to), one can run the following JS code snippet: 
try {
	var memory = new WebAssembly.Memory({initial:2**16});
	console.log('4gb wasm memory: supported');
} catch(e) {
	console.error(e);
}
